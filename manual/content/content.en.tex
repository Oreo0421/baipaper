So far we have only set up the template, but have not yet included any content
into the project. This chapter is intended to provide orientation for the
structure of a scientific thesis. We highlight topics that are relevant for all
types of theses: e.g. outline, bibliographies, formal aspects and language.
Of course, these recommendations do not have to be suitable for all academic
disciplines. If other regulations apply in your field, then give them priority
in all cases.

While this chapter deals with obligatory components of a scientific thesis,
there are many other applications for which \LaTeX\ offers support.
A collection of case studies and best practices has been in appendix~\ref{app:latex}.
Please check which of the sections in this appendix may be relevant to you.
