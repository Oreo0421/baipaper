With the package~\ctanref{subcaption}, illustrations can be subdivided by an
additional numbering level. The packages~\ctanpkg{subfigure} and \ctanpkg{subfloat}
perform a similar function. The package~\ctanpkg{subcaption} is newer.
Listing~\ref{lst:latex:fig:subcaption} shows an example of the use of the package~\ctanpkg{subcaption}.
The result is shown in Figure~\ref{fig:latex:fig:subcaption}. The \texttt{subcaption}
environment has the same function as the \texttt{minipage} environment.
