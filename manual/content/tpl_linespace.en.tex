As a rule, single line spacing should be used. In order to provide space for
the reviewer's comments, the line spacing can also be increased to
1\nicefrac{1}{2} times the line spacing. However, if the line spacing is
spacing can quickly give the impression that space is to be "wasted" in this
way. The setting of the line spacing is done by the package~\ctanref{setspace}
according to Listing~\ref{lst:tpl:linespace}.

The package~\ctanpkg{setspace} ensures that the single line spacing is used in
footnotes and floating environments. If you want to use extended line spacing
is to be used, the template under \texttt{latex/linespace\_mod.tex} provides
the corresponding \TeX code.
